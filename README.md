# YTBlog Theme

一个为 Typecho 设计的互联网科技风主题，特点是：

- 首页 Hero 自动抓取最近带图文章并轮播展示
- 顶部菜单自动展示分类，并支持二级分类下拉
- 整体简约、扁平化、卡片式布局
- 适合技术博客、产品博客、工程笔记
- 响应式布局，兼容桌面与移动端
- 评论关闭后仍可展示历史评论
- 无图文章自动回退到默认占位图，支持后台配置图片 API

## 文件说明

- `functions.php`：主题配置和文章自定义字段
- `index.php`：首页，包含大图 Hero 和文章流
- `archive.php`：归档、分类、标签、搜索结果页
- `post.php`：文章详情页
- `page.php`：独立页面模板
- `comments.php`：评论列表和表单
- `assets/css/style.css`：主题样式
- `assets/js/theme.js`：移动导航、头部滚动状态和 Hero 轮播控制

## 使用方式

1. 将整个目录放入 Typecho 的 `usr/themes/` 下。
2. 登录 Typecho 后台，在“控制台 -> 外观”中启用 `YTBlog Theme`。
3. 在“外观设置”中可填写站点 Logo 文案、页脚文案，以及无图文章使用的占位图 / 图片 API。
4. 顶部菜单会自动显示首页、分类、二级分类、独立页面等入口。
5. 首页 Hero 会自动读取最近几篇带图文章并轮播：
   - 优先使用 `cover`
   - 如果没有 `cover`，则读取正文里的第一张图片
   - 如果最近文章都没有图片，首页不显示 Hero
6. 首页会额外输出“重点内容”和“推荐文章”区块。
7. 在文章编辑页可额外填写：
   - `cover`：文章封面图
   - `summary`：卡片摘要
   - `badge`：卡片左上角角标
8. 如果文章关闭了新评论，但已有历史评论，主题仍会保留评论展示，并提示“评论已关闭”。
9. 如果文章没有封面图，主题会自动使用默认占位图；你也可以在主题设置里填写自己的图片 API。

## 建议

- Hero 建议使用宽度 `1600px` 以上的横向图片。
- 封面图建议保持统一比例，以获得更整洁的卡片视觉。
